# Convert any images to webP image in batch.
---

## Instruction

- Setup
  - Clone this project
  - Relative path to App.js in the console run 'npm install'

- Execution
  - Place all images in the **_images_** folder
  - Relative path to App.js, in the console run 'npm start'

That's it!
